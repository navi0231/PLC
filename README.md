# Sequential_Motor_Control_System
​1. 项目概述

​本程序基于 西门子 TIA Portal V20 与 S7-1500 PLC 开发，实现了三台电动机（M1、M2、M3）的顺序启动与顺序停止功能。

程序采用单次流程逻辑，在任务完成后能够自动清理中间状态并保持待机，无需重复重启 PLC 即可再次运行。



​2. 控制要求

​顺序启动流程：
​按下 I0.4 (启动按钮)：M1 立即启动。
​延时 2秒：M2 自动启动。
​再延时 2秒：M3 自动启动。

​顺序停止流程：
​按下 I0.5 (停止按钮)：M3 立即停止。
​延时 5秒：M2 自动停止。
​再延时 1.5秒：M1 自动停止。



​3. 技术核心实现

​初始化逻辑 (First Scan)：利用系统变量 M1.0 (FirstScan) 在 PLC 上电瞬间激活初始步 M2.0，确保系统进入“预备启动”状态。

​步进状态机：通过中间位 (M2.1 ~ M2.6) 严格控制流程节点，确保动作的唯一性和准确性。

​批量复位 (RESET_BF)：在流程结束（T3 定时到）时，通过 RESET_BF 指令一键清除所有中间过程位（M2.1 ~ M2.6），但特意保留初始步 M2.0 的接通状态，实现了无需重启即可重复操作的便利性。

​单次运行优化：通过对跳转条件的精准控制，解决了 M1 在结束瞬间可能被误触发重启的问题，完全符合教材对单次流程的要求。
